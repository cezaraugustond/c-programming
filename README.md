# Programação em C

Este repositório contém meus estudos, exercícios e implementações práticas em linguagem C, abrangendo tanto a formação acadêmica quanto o aprendizado autodidata.

---

## Estrutura do Repositório

Abaixo estão as principais seções deste repositório, organizadas de acordo com a origem dos estudos e o conteúdo abordado.

---

### [Independent Studies](./independent-studies)

Conteúdo desenvolvido a partir de estudos independentes, com base em aulas e materiais do canal [Programação Descomplicada](https://www.youtube.com/@progdescomplicada).

Você encontrará:

* Fundamentos da linguagem C
* Estruturas de dados
* Algoritmos de ordenação
* Exercícios práticos

Objetivo: consolidar a base teórica e prática por meio de aprendizado autodidata.

---

### [UFS - Programação Imperativa](./ufs-imperative-programming)

Conteúdo desenvolvido durante a disciplina de Programação Imperativa na Universidade Federal de Sergipe (UFS).

Você encontrará:

* Exercícios propostos em aula
* Práticas de laboratório
* Aplicação dos conceitos fundamentais da programação

Foco: construção da base lógica e domínio dos conceitos iniciais da programação.

---

### [UFS - Projeto e Análise de Algoritmos](./ufs-algorithms-analysis)

Conteúdo desenvolvido durante a disciplina de Projeto e Análise de Algoritmos na UFS.

Você encontrará:

* Exercícios de análise de algoritmos
* Estudos sobre complexidade
* Implementações práticas de algoritmos

Foco: desenvolvimento da capacidade de análise e otimização de soluções.

---

### [UFS - Estrutura de Dados](./ufs-data-structures)

Conteúdo desenvolvido durante a disciplina de Estrutura de Dados na UFS.

Você encontrará:

* Implementações de estruturas de dados
* Exercícios práticos
* Aplicações de listas, pilhas, filas, árvores, grafos e mais

Foco: organização e manipulação eficiente de dados.

---

## Observações

* Este repositório está em constante evolução conforme avanço nos estudos.
* Os conteúdos estão organizados para facilitar a navegação e demonstrar progressão no aprendizado.

---

## Objetivo

Construir uma base sólida em programação em C, com foco em:

* Lógica de programação
* Estruturas de dados
* Algoritmos
* Boas práticas de desenvolvimento
